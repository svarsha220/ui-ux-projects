# Poster Design :
The poster was crafted using Canva, focusing on a futuristic aesthetic to align with the event’s technological and sustainable theme.

<h4>Design Process</h4>

* ***Theme Selection***: A futuristic background to symbolize innovation and sustainability.
* ***Typography***: Modern fonts that enhance readability and create a visual hierarchy.
* ***Color Palette***: Deep purples and golds to evoke sophistication and align with the theme.
* ***Iconography***: Minimalistic icons and structured layouts to ensure clarity and visual appeal.

# Mascot Design :

The objective was to design an original character that embodies the theme and values of a specific event, while also showcasing the mascot in various poses and contexts that are visually appealing mascot that can serve as a central character for branding, promotions, and engagement in an event or campaign.

<h4>Design Elements : </h4>
<h5>Base Mascot Design</h5>
The mascot features a simple, friendly, and expressive design to evoke relatability and appeal.
The character’s rounded form and minimalistic style make it versatile for various contexts.

<h5>Different Poses and Expressions</h5>:
The mascot is showcased in multiple poses, such as standing, jumping, and waving.
Includes facial expressions to highlight different emotions (e.g., happy, surprised, sad).
